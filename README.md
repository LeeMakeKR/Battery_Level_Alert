# 배터리 레벨 알림 시스템 (Battery Level Alert)

리튬이온 배터리 5셀의 전압을 모니터링하여 배터리 상태를 부저로 알려주는 Arduino 기반 시스템입니다.

## 📋 프로젝트 개요

이 시스템은 리튬이온 배터리 팩(5셀)의 전압을 실시간으로 모니터링하고, 배터리 잔량에 따라 5단계로 구분하여 부저를 통해 사용자에게 알림을 제공합니다. 절전 모드를 사용하여 전력 소모를 최소화합니다.

## ⚡ 주요 기능

### 전압 단계별 알림
- **1단계 (18.9V 이상)**: 부저 1번 울림
- **2단계 (17.9V ~ 18.9V)**: 부저 2번 울림  
- **3단계 (16.15V ~ 17.9V)**: 부저 3번 울림
- **4단계 (15.0V ~ 16.15V)**: 부저 4번 울림
- **5단계 (15.0V 미만)**: 연속 부저 울림 (위험 상태)

### 절전 기능
- 3분 간격으로 전압 측정
- 측정 후 자동으로 절전 모드 진입
- 워치독 타이머(WDT)를 사용한 저전력 운영
- 5단계에서는 절전 모드 없이 연속 경고

### 정밀한 전압 측정
- 외부 기준 전압 사용 (LM4040-2.5V)
- 전압 분배 회로를 통한 안전한 측정
- 10비트 ADC 분해능으로 정확한 측정

## 🔄 버전 업데이트 계획

### 하드웨어 개선
- **전력 효율성 향상**: TPS62177 DC-DC 컨버터 사용 검토
- **소형화**: ATTiny85 칩 사용 고려
- **입력 전압 범위 확장**: 20V 이상 입력 지원

### 소프트웨어 개선
- ATTiny85용 코드 최적화
- 소프트웨어 시리얼 통신 추가
- 더 정밀한 절전 모드 제어
